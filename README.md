# SBB Precipitation & Train Delay Study

Does rain make Swiss trains late? This project investigates the correlation between precipitation and arrival delays at **Zürich HB**, **Basel SBB**, and **Bern** using SBB and MeteoSwiss data from 2022–2025.

🔗 **[Live App](https://bfh-bina-project-r5jxe6szjtrjmkz7fnq9ae.streamlit.app)** · **[Analysis Notebook (Colab)](https://colab.research.google.com/github/HaKePlan/bfh-bina-project/blob/main/notebooks/case_study.ipynb)**

---

## Key Findings

- **4.1M train connections** analysed over 45 months
- Spearman correlation ρ = 0.074 — statistically significant but very weak
- The effect is a **threshold effect** (dry vs. wet), not linear
- Best model: Random Forest (RMSE 6.61, R² ≈ 0.000) — precipitation alone explains < 0.25% of delay variance

## Setup

```bash
# 1. Clone and configure
git clone <repo> && cd bfh-bina-project-wt1
cp .env.example .env          # edit with your DB credentials

# 2. Start PostgreSQL
docker-compose up -d

# 3. Install dependencies
pip install -r scripts/requirements.txt

# 4. Load data
python -m scripts.load_meteo
python -m scripts.collect_sbb --start-year 2024 --end-year 2025

# 5. Run the notebook
jupyter notebook notebooks/analysis.ipynb
```

> **Quick test:** add `--months 1,2` to process only Jan–Feb.

---

## Web App

The Streamlit app predicts delays from real MeteoSwiss 7-day forecasts. It needs only the trained model — no database.

```bash
# Local (run from project root)
streamlit run app/app.py

# Docker
docker build -f app/Dockerfile -t sbb-delay-app .
docker run -p 8501:8501 sbb-delay-app
```

**Prerequisite:** `models/model_all_data.pkl` must exist (generated by notebook Section 4).

## ⚠️ Known Limitations

- Single predictor (precipitation) — confounders like holidays, incidents, and season are not modelled
- MeteoSwiss station may not align with the actual train route
- Overnight trips excluded
- Hourly forecast ÷ 6 approximates the 10-min model input
